# 🍔 Food Delivery Website

A modern and responsive Food Delivery Landing Page built using **HTML5** and **CSS3**.

This project showcases a clean user interface with multiple sections including Hero Banner, Food Menu, How It Works, App Download, Subscription, and Footer.

---

## 🚀 Features

- Responsive Design
- Sticky Navigation Bar
- Modern Hero Section
- Search Box UI
- How It Works Section
- Food Menu Cards
- App Download Section
- Subscription Section
- Footer with Contact Information
- Smooth Hover Effects
- Mobile Friendly Layout

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
